# Quotes ETL Pipeline

This project is a small ETL pipeline built in Python

## What it does

- Extract quotes, authors, and tags from quotes.toscrape.com
- Transforms and clean data
- Remove dupliocate quotes
- Loads the final data into CSV and SQLite

## Tech Stack

- Python
- BeautifulSoup
- requests
- pandas
- sqlite

## How to run

```bash
python main.py
```
